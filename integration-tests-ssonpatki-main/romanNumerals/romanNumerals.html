About Roman Numerals

Roman numerals are an ancient number system that originated in Rome. The modern system uses combinations of letters from the Latin alphabet to represent values.

  • I = 1
  • V = 5
  • X = 10
  • L = 50
  • C = 100
  • D = 500
  • M = 1000

"Old" style uses only additive notation, while "modern" style uses subtractive notation.

  • 4 = IIII (old), IV (modern)
  • 9 = VIIII (old), IX (modern)
